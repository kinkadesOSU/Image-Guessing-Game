Image Matching Game

Instructions

Select the number of squares you would like to play with. The page will then load. The game is ready to play when you see "Images Loaded. Start Playing!" in a green box. You may now start picking images two at a time. If the images are a match, they will visible. If they are not a match, they will flip back over and you can try again. The objective of the game is to find all pairs of images in as few tries as possible.
